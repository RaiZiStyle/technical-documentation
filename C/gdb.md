# Command line debugging with GDB

`break` : define a break point (can be function)
`run` : run the program
`next` : step over (`n`)
`tui enable` : enable text user interface (graphic)
`p`: print variable
`display` : display variable after each step



Show pointer as an array : `p *pointer@size` (ex: `p *ptr@10` to show 10 elements of the array pointed by ptr)
