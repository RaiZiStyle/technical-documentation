- [typesafe_inheritance.c](typesafe_inheritance.c): I haven't really looked into it yet. Taken from "http://www.deleveld.dds.nl/inherit.htm".
- [Duffs_device](Duffs_device.c) : I haven't really looked into it yet. Taken from "https://en.wikipedia.org/wiki/Duff%27s_device".
- [nob.h](nob.h): No-build system from "https://github.com/tsoding/nob.h".
- [nob.c](nob.c): Compiles every C file in this directory.
